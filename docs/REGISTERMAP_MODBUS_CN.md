# SEN07xx 智能气体：Modbus RTU 寄存器表

本文档按「类型 | 地址 | 名称 | 读写 | 数据/范围 | 典型值 | 说明」整理。  
**规范来源**：`SEN0738/src/interfaces/RtuRegisterMap.h` 与 Arduino 库 `DFRobot_IntelligentGasSensor.h`。

> 部分通用 Modbus 表里「输入 0x0000=VID、0x0001=PID」。**本产品线为 0x0000=PID、0x0001=VID**。

---

## 一、功能码

| 类型 | 功能码 | 说明 |
|------|--------|------|
| 输入寄存器 | **0x04** | 只读；每寄存器 16 bit |
| 保持寄存器 | **0x03** 读 / **0x06** 单写 / **0x10** 多写 | 配置与 COMMIT |

**32 位浓度**：**0x0009（低字）+ 0x000A（高字）**，小端 32 位整数语义。

---

## 二、输入寄存器（FC 0x04）

共 **18** 个（`0x0000`～`0x0011`）。

| 类型 | 地址 | 名称 | 读写 | 典型值 | 说明 |
|------|------|------|------|--------|------|
| 输入 | 0x0000 | PID | R | 0xC746 | RS-485 板；TTL 为 0xC742 |
| 输入 | 0x0001 | VID | R | 0x3343 | DFRobot |
| 输入 | 0x0002 | Modbus 从站地址 | R | 1～247 | 当前从站号 |
| 输入 | 0x0003～0x0004 | 保留 | R | 0 | |
| 输入 | 0x0005 | 寄存器图版本 | R | 0x0100 | |
| 输入 | 0x0006 | 板型 | R | 2 | 2=RS-485 |
| 输入 | 0x0007 | 状态 | R | — | bit0=测量有效 |
| 输入 | 0x0008 | GAS_CODE | R | — | 主机库译码类型/单位 |
| 输入 | 0x0009 | 浓度低字 | R | — | |
| 输入 | 0x000A | 浓度高字 | R | — | |
| 输入 | 0x000B | 小数位 | R | — | 浓度 = 原码 × 10^−dp |
| 输入 | 0x000C～0x0011 | 年月日时分秒 | R | — | 无 RTC 时可全 0 |

---

## 三、保持寄存器（FC 0x03 / 0x06 / 0x10）

共 **8** 个（`0x0000`～`0x0007`）。改**波特率/校验/从站**后写 **0x0007 = 0xA501** COMMIT 落盘。**0x0005 探头休眠不入 EEPROM**，写后即时生效。

| 类型 | 地址 | 名称 | 读写 | 典型值 | 说明 |
|------|------|------|------|--------|------|
| 保持 | 0x0000 | 保留 | R/W | 0 | 读恒 0 |
| 保持 | 0x0001 | 保留 | R/W | 0 | 读恒 0（无 UART 主动上报） |
| 保持 | 0x0002 | 保留 | R/W | 0 | |
| 保持 | 0x0003 | 波特率编码 | R/W | 3 | 1=2400 … 8=115200 |
| 保持 | 0x0004 | 校验/停止位 | R/W | 0 | 8N1 等 |
| 保持 | 0x0005 | 探头休眠 | R/W | 0 | **0**=运行（CS 高），**1**=休眠（CS 低）；立即生效，**不落盘**；USB 可 `sensor sleep`/`sensor wake` |
| 保持 | 0x0006 | 从站地址 | R/W | 1 | 写后立即用于应答 |
| 保持 | 0x0007 | COMMIT | W | 0xA501 | EEPROM 落盘 |
